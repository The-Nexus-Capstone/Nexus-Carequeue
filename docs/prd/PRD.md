# Product Requirements Document (PRD)

## Product: CareQueue
## Version: 1.0 (MVP)
## Date: January 2025

## Overview
Queue management system for public health clinics.

## Scope
Two-clinic deployment for capstone project.

## Features
1. Patient queue joining
2. SMS notifications
3. Wait time estimation
4. Staff queue management
5. Real-time updates

## Success Metrics
- Reduce average wait time
- Improve patient satisfaction
- Enable remote queue joining
